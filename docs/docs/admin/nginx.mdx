---
title: How to use Nginx' auth_request
---

Anubis supports using Nginx' auth_request module. The following config snippet serves as an example on how to integrate it into your Nginx config.

```nginx
location /.within.website/ {
    proxy_pass http://127.0.0.1:8923;
    auth_request off;
}
location @redirectToAnubis {
    return 307 /.within.website/?redir=$scheme://$host$request_uri;
    auth_request off;
}
location / {
    auth_request /.within.website/x/cmd/anubis/api/check;
    error_page 401 = @redirectToAnubis;
}
```
